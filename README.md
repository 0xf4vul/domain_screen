# webInfoScan

网站信息批量采集脚本。


[bcScan](https://github.com/TheKingOfDuck/bcScan)的升级版本，包括但不限于提升了截图的速度，报告的友好度。

py3环境下运行，报告生成在report目录下。依赖缺啥就自己pip装啥就OK了。
